If you have a Rockto account, this plugin is what you need to connect them with your blog! When you publish a new wordpress article, the Rockto Autoshare plugin will share on your Rockto Account directly.

== Description ==

Instead of manually post to **Rockto** with every new post you publish on your blog, this plugin does it for you.   
**The plugin requires Php4 or higher, Curl Libraries.**

== Installation ==

1.   Unzip and upload 'Rockto Autoshare' directory to your '/wp-content/plugins/' directory   
2.   Activate the plugin through the 'Plugins' menu in the Admin panel  
3.   Go to the plugin setting page to enable and tune it        
4.   Ensure the **rocktoSessionData.txt** file, in the **/wp-content/plugins/rockto-auto-share/** directory is writable from PHP
